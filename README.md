How to use
------------------

```html

<html>
<head>
    <script src="phoneinput.js"></script>
</head>
<body>
    <input type="tel" data-tel-input placeholder="Phone" maxlength="18" />
</body>
</html>
```

JS code in `phoneinput.js` will find all `input` with `data-tel-input` attribute and process it.
